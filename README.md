# AsyncHire

AsyncHire is a simple async video interview platform that helps teams interview candidates without scheduling live calls.

Candidates record video answers to predefined questions at their own time, and employers review responses when it suits them.

## Why AsyncHire?
- No scheduling back and forth
- Interview candidates on their own time
- Simple and focused on async hiring only

## MVP Features
### Employer
- Create async interviews with custom questions
- Share interview links with candidates
- View candidate video submissions in one dashboard

### Candidate
- Access interview via a link (no login required)
- Record video responses per question
- Preview and re-record before submitting
- Submit interview once

## Planned Tech Stack
- Next.js
- Tailwind CSS
- Node.js / API Routes
- MongoDB
- Cloud-based video storage

## Status
This project is currently in early development and is being built as both a real-world product and a portfolio project.
